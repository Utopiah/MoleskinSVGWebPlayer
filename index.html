SVG web player for hand drawing devices

⚠️ Testing is done live so... if things stopped working it's normal ;)⚠️

Purpose

Drawing helps to think about complex system. As we draw we find link and become able to summarize. Unfortunately the final drawing is often incomprehensible for others because the explanations that was done with the drawing is gone. As importantly the chronology of the drawing is also lost. The goal of thise player is to enable replaying of chronology and efficient sharing.

Supported devices

  • Moleskine
  • Surface Hub (some rainbow coloring lost)
  • reMarkable (not pencil style)
  • visual timeline editor used to define bookmarks or checkpoints
  • media synchronisation used to provide a video or audio track
